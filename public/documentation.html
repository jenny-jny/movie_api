myFlix is the server-side component of a “movies” web application. This API will provide users with access to information from a database about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Request URL HTTP method Request body data format Response body data format
Return a list of all movies to the user /movies GET None A JSON object holding data about all the movies
Return data (description, genre, director, image URL) about a single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie, containing title, director, and genre property. Example: { title: "Casablanca", director: "Michael Curtiz", genre: "Romance", imageUrl: "https://en.wikipedia.org/wiki/Casablanca_(film)#/media/File:CasablancaPoster-Gold.jpg" }
Return data about a genre (description) by movie title /movies/:title/genre GET None A JSON object holding data about a single genre. Example: { genre: "Romance" }
Return data about a director (bio, birth year, death year) by title and director's name /movies/:title/:director GET None A JSON object holding data about a single director, containing bio, birth year, and death year. Example: { director: "Michael Curtiz", bio: "A Hungarian-born American film director, recognized as one of the most prolific directors in history.", birthYear: "1886", deathYear: "1962" }
Add a new user (register) /users POST A JSON object holding data about the new user to add, structured like: { "userEmail": "jenny.yu.0414@gmail.com", "username": "jenny_jny" } A JSON object holding data about the new user that was added
Update a user's info (username) by the user's email /users/:userEmail/:username PUT None A text message indicating the user's username was updated
Add a movie to the user's list of favorites /users/:username/favorites POST A JSON object holding data about the movie to add, structured like: { "title": "The Wolf of Wall Street", "director": "Martin Scorsese", "genre": "Comedy" } A text message indicating the user's favorite movie was added to their list of favorites
Remove a movie from the user's list of favorites /users/:username/favorites/:movie DELETE None A text message indicating the movie is removed from the user's list of favorites
Remove an existing user by the user's email (deregister) /users/:userEmail DELETE None A text message indicating the user's email is removed
