myFlix is the server-side component of a “movies” web application. This API will provide users with access to information from a database about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Request URL HTTP method Request body data format Response body data format
Return a list of all movies to the user /movies GET None A JSON object holding data about all the movies
Return data (title, description, genre, director, image URL, featured) about a single movie by title to the user /movies/:Title GET None A JSON object holding data about a single movie, containing title, director, and genre property.
Return data about a genre (description) by movie title /movies/:title/genre GET None A JSON object holding data about a single genre.
Return data about a director (name, bio, birth year, death year) by title and director's name /movies/:title/:director GET None A JSON object holding data about a single director, containing name, bio, birth year, and death year.
Add a new user (register) /users POST A JSON object holding data about the new user to add, structured like: { "Username": "jenny_jny", "Password": "12345", "Email": "ranch@gmail.com", "Birthday": "1967-01-01" } A JSON object holding data about the new user that was added, structured like: { "FavoriteMovies": [], "_id": "8hrjksghda84324", "Username": "jenny_jny", "Password": "12345", "Email": "ranch@gmail.com", "Birthday": "1967-01-01T00:00:00.000Z", "__v": 0 }
Update a user's info (username) by the user's email /users/:email/:username PUT None A text message indicating the user's username was updated
Add a movie to the user's list of favorites /users/:Username/favorites/:MovieID POST A JSON object holding data about the movie to add, structured like: { "FavoriteMovies": "MovieID" } A text message indicating the user's favorite movie was added to their list of favorites
Remove a movie from the user's list of favorites /users/:Username/favorites/:MovieID DELETE None A text message indicating the movie is removed from the user's list of favorites
Delete an existing user by username (deregister) /users/:Username DELETE None A text message indicating the user's email is removed
